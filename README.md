# Homeschool Dashboard (5th Grade)

A simple, no-login dashboard for planning and tracking a 5th grade homeschool week.

- Hosted on GitHub Pages (opens the app directly)
- Checkboxes + notes saved in browser localStorage
- Easy to customize the weekly template

## Customize
Edit `weekTemplate` and `subjects` in `app.js`.

## Privacy
No accounts, no analytics, no server. Everything is stored locally in the browser.
